# SignalR Real-Time Chat

> Real-time bidirectional communication with WebSockets.

## Features
- **SignalR Hub** - Server-side hub for message broadcasting
- **WebSocket** - Persistent connection for real-time updates
- **Broadcast** - Send to all connected clients
- **Connection Management** - Join/leave notifications

## Run
```bash
cd samples/06-CuttingEdge/SignalRExample
dotnet run
# Open: http://localhost:5000 in multiple browser tabs
```

## Use Cases
- Chat applications
- Live dashboards
- Real-time notifications
- Collaborative editing
- Gaming (multiplayer state sync)

**Performance:** 100,000+ concurrent connections per server!
