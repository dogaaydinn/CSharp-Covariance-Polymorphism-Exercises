# Blazor Full Stack

> Interactive web UI with C# - no JavaScript required!

## Features
- **Blazor Server** - Real-time UI updates via SignalR
- **Component-based** - Reusable Razor components
- **Two-way binding** - `@bind` for forms
- **Web API** - REST endpoints + Blazor UI

## Run
```bash
cd samples/06-CuttingEdge/BlazorFullStack
dotnet run
# Navigate to: http://localhost:5000
```

## How It Works
- Frontend: Blazor components (`App.razor`)
- Backend: ASP.NET Core API (`Program.cs`)
- Real-time: SignalR connection for UI updates
- No JavaScript: All code in C#!

**Perfect for:** Full-stack C# developers who want to avoid JavaScript frameworks.
