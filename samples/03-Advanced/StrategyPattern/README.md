# Strategy Pattern

**Konu**: Behavioral pattern - Runtime algorithm selection

## Öğrenilen Kavramlar
- ✅ Strategy interface
- ✅ Concrete strategies
- ✅ Context class
- ✅ Runtime algorithm switching

## Kullanım
```bash
dotnet run
```

## Ne Zaman Kullanılır?
- Aynı işi farklı şekillerde yapmak gerekiyorsa
- if/else zinciri çok uzunsa
- Runtime'da algorithm değişecekse

## Örnekler
- Payment methods (Credit, PayPal, Crypto)
- Shipping options (Standard, Express, Overnight)
- Sorting algorithms (QuickSort, MergeSort, BubbleSort)
- Compression (ZIP, RAR, 7Z)
