# Decorator Pattern

**Konu**: Structural pattern - Add responsibilities dynamically

## Öğrenilen Kavramlar
- ✅ Component interface
- ✅ Concrete component
- ✅ Decorator base class
- ✅ Concrete decorators

## Gerçek Dünya
- I/O Streams (BufferedStream, GZipStream)
- Middleware (ASP.NET Core)
- UI components (borders, scrollbars)
