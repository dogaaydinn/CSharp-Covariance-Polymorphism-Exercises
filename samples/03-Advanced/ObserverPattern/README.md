# Observer Pattern

**Konu**: Behavioral pattern - One-to-many dependency

## Öğrenilen Kavramlar
- ✅ Subject/Observable
- ✅ Observer
- ✅ Push vs Pull model
- ✅ C# Events (modern implementation)

## Kullanım
```bash
dotnet run
```

## Ne Zaman Kullanılır?
- State değişikliği birden fazla nesneyi ilgilendiriyorsa
- Loose coupling gerekiyorsa
- Event-driven architecture

## Gerçek Dünya
- Stock market updates
- Newsletter subscriptions
- UI event handling
- Pub/Sub systems
