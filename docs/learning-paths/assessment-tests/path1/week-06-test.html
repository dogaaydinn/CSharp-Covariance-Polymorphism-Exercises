Short Answer (4.5 pts)

  1. (1.5 pts) Write LINQ to group products by category and get count per category.

  2. (1.5 pts) How would you calculate average price per category using GroupBy?

  3. (1.5 pts) Explain the difference between ToDictionary() and ToLookup().

Code Analysis (1.5 pts)

  1. Complete this code to get top 3 categories by total sales:
var result = sales
    .GroupBy(s => s.Category)
    // Your code here

Answer Key

  1. b | 2. c | 3. b | 4. c | 5. a

  2. products.GroupBy(p => p.Category).Select(g => new { Category = g.Key, Count = g.Count() })

  3. products.GroupBy(p => p.Category).Select(g => new { Category = g.Key, AvgPrice = g.Average(p => p.Price) })

  4. ToDictionary() throws on duplicate keys, one value per key. ToLookup() allows multiple values per key, no exception.

.Select(g => new { Category = g.Key, TotalSales = g.Sum(s => s.Amount) })
.OrderByDescending(x => x.TotalSales)
.Take(3)
