Short Answer (4.5 pts)

  1. (1.5 pts) Write LINQ to get products where price > 50, ordered by name, select only names.

  2. (1.5 pts) Explain the difference between First() and FirstOrDefault(). When would each throw an exception?

  3. (1.5 pts) What are anonymous types in LINQ? Give an example.

Code Analysis (1.5 pts)

  1. What's the output and why?
var numbers = new List<int> { 1, 2, 3 };
var query = numbers.Where(n => n > 1);
numbers.Add(4);
numbers.Add(5);
foreach (var n in query) Console.Write(n + " ");

Answer Key

  1. a | 2. b | 3. c | 4. c | 5. c

  2. products.Where(p => p.Price > 50).OrderBy(p => p.Name).Select(p => p.Name)

  3. First() throws if empty, FirstOrDefault() returns default (null/0). Use First when data must exist, FirstOrDefault when it might not.

  4. Anonymous types are compiler-generated types for projections: select new { p.Name, p.Price } creates type with Name and Price properties.

  5. Output: 2 3 4 5 - Deferred execution means query sees all numbers including newly added 4 and 5 when enumerated.
