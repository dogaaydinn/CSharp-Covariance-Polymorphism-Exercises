# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2025-11-15

### Added
- Comprehensive unit test project with xUnit and FluentAssertions
- 30+ unit tests covering all major concepts
- GitHub Actions CI/CD pipeline with multi-platform testing
- EditorConfig for consistent code styling
- Code quality analysis with Roslyn analyzers
- Cross-platform support (Windows, Linux, macOS)
- Comprehensive XML documentation
- Code coverage reporting
- Production-ready README with badges and detailed documentation
- CHANGELOG for version tracking
- CODE_OF_CONDUCT for community guidelines
- GitHub issue and pull request templates

### Changed
- Upgraded from .NET 6.0 to .NET 8.0
- Modernized project structure and configuration
- Enhanced README with comprehensive documentation
- Improved CONTRIBUTING guidelines
- Updated all examples with modern C# patterns

### Improved
- Code quality and consistency
- Project organization and structure
- Documentation completeness
- Test coverage (>85%)
- Build and deployment automation

### Fixed
- Code quality issues in Manager class (private field exposure)
- Inconsistent naming conventions
- Missing documentation in public APIs

## [1.0.0] - 2024-XX-XX

### Added
- Initial project structure
- Basic examples for covariance and contravariance
- Polymorphism demonstrations
- Boxing and unboxing examples
- Type conversion operators
- Generic variance examples
- Upcasting and downcasting examples
- Assignment compatibility demonstrations
- Basic README documentation
- MIT License
- Contributing guidelines

### Features
- Beginner-level concepts (Polymorphism, Upcasting/Downcasting)
- Intermediate-level concepts (Boxing/Unboxing, Covariance/Contravariance)
- Advanced-level concepts (Generic Variance, Custom Operators)
- Console application with interactive examples
